# Security Policy

## Supported Versions

We currently support the latest release of Basic Foundation. Make sure you are using an up-to-date version to receive
security patches.

## Reporting a Vulnerability

If you discover a security vulnerability, please contact us immediately. Do not publicly disclose the issue until it has
been addressed.

To report a vulnerability, please email [paul@basicmachines.co] with details, including reproduction steps and any
relevant code or logs.
